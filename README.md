## 🚀 Overview
The **IPL 2026 Winner Prediction Engine** is a high-performance analytics platform designed to forecast the outcomes of the 2026 Indian Premier League. Unlike traditional models, this system synchronizes with **Live 2026 Season Standings** to provide minute-by-minute probability shifts and championship projections.

## 🧠 Technical Architecture
- **Backend:** Flask (Python) handles the real-time data synchronization and predictive logic.
- **Frontend:** Premium Landing Page with CSS 3D Glassmorphism and Responsive Design.
- **Data Foundation:** 
    - Historical Training: 2008 - 2025 Ball-by-Ball IPL Dataset.
    - Live Sync: Real-time 2026 points table integrations and performance momentum tracking.
- **Algorithm:** Random Forest Classifier optimized for "Clutch Factor" and Venue influence.

## ✨ Key Features
- **Live 2026 Leaderboard:** Real-time synchronization of the top 5 contenders.
- **Predictive Depth:** Detailed analysis of team strengths (Squad Alpha, Bowling DNA, Historical Pivot).
- **The Challengers:** Survival math for teams currently outside the playoff bracket.
- **Dynamic UI:** Real-time timestamps and automatic refresh logic.

## 🛠️ Local Setup
1. **Clone the repository.**
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Start the Engine:**
   ```bash
   python app.py
   ```
4. **Access the Dashboard:**
   Navigate to `http://127.0.0.1:5000`

## 🌍 Deployment
This project is ready for production hosting on platforms like **Render**, **Heroku**, or **DigitalOcean**. 
The `Procfile` is configured to use `gunicorn` for a high-concurrency production environment.
