## 🚀 Overview
The **IPL 2026 Winner Prediction Engine** is a high-performance analytics platform designed to forecast the outcomes of the 2026 Indian Premier League. The system features a built-in **Season Simulation Engine** that automatically evolves the standings and championship math every day until the Grand Finale on May 24, 2026.

## 🧠 Technical Architecture
- **Backend:** Flask (Python) with a custom Season Simulation Engine.
- **Frontend:** Premium Landing Page with CSS 3D Glassmorphism and Real-Time data binding.
- **Live Points Table:** Automated tracking for all 10 IPL teams (P, W, L, NRR, Pts).
- **AI Model:** Random Forest Classifier trained on 18 seasons of IPL data (2008-2025).

## ✨ Key Features
- **Live Points Table:** A dedicated real-time section tracking match-by-match progression for all 10 franchises.
- **Automated Season Progression:** Standings automatically "age" day-by-day towards the final.
- **3D Interactive Rankings:** Perspective-based team cards for the Elite Bracket.
- **The Challengers:** Survival probability indexing for mid-to-lower bracket teams.
- **Championship Verdict:** Final AI proclamation of the projected 2026 winner.

## 🛠️ Local Setup
1. **Clone the repository.**
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Start the Engine:**
   ```bash
   python api/index.py
   ```
4. **Access the Dashboard:**
   Navigate to `http://127.0.0.1:5000`
