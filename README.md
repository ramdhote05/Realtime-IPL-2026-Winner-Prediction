## 🚀 Overview
The **IPL 2026 Winner Prediction Engine** is a high-performance, autonomous analytics platform designed to forecast the outcomes of the 2026 Indian Premier League. The system features a **Zero-Maintenance Season Progressor** that automatically evolves the standings and championship probability every day.

## 🧠 Technical Architecture
- **Backend:** Flask (Python) with a date-aware Season Simulation Engine.
- **Automation:** Smart `datetime` logic ensures the leaderboard "ages" automatically every 24 hours.
- **Frontend:** Premium Landing Page with CSS 3D Glassmorphism and Real-Time data binding.
- **Live Points Table:** Automated tracking for all 10 IPL franchises (P, W, L, NRR, Pts).
- **AI Model:** Random Forest Classifier trained on 18 seasons of historical IPL data (2008-2025).

## ✨ Key Features
- **Zero-Maintenance Sync:** Standings automatically progress day-by-day towards the Grand Finale on May 24, 2026.
- **Full Points Table:** A dedicated real-time section tracking match-by-match metrics for all 10 teams.
- **Fixtures Center:** Displays the official upcoming 2026 match schedule sourced from live data.
- **Contextual Analytics:** Integrated "How, Why, and When" descriptions for all analytical sections.
- **Professional Branding:** Trademarked prediction engine footer with linked developer profile.
